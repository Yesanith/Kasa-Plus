# KASA+

KASA+ is a money counting and safe management application built with Flutter.

## Features

*   **Money Counting:** Quickly count cash with denomination inputs for TRY, USD, and EUR.
*   **Safe Management:** Track your safe inventory ("Kasa") automatically.
*   **History:** View past records of your counts.
*   **Statistics:** Visualize your net total and differences over time with charts.
*   **Reconciliation:** Calculate differences between counted cash, initial cash, and target amounts.
*   **Multi-Currency:** Support for Turkish Lira (TRY), US Dollar (USD), and Euro (EUR).
*   **Dark Mode:** Full support for system-wide dark mode with a sleek emerald/dark theme.
*   **Localization:** English and Turkish language support.

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get`.
3.  Run `flutter run`.

## Architecture

*   **State Management:** Provider
*   **Persistence:** Shared Preferences
*   **Charts:** fl_chart
*   **Localization:** flutter_localizations
